# pet-info-management
The Pet Management System is a project developed in C programming language. 

The main aim of this project is to store and manage details of different pets such as their name, age, breed, and health information. 

In this project, I used concepts like functions, strings, arrays, and file handling to perform various operations such as adding, viewing, updating, and deleting pet records. 

This project helps to understand how data can be stored and managed efficiently using files in C without using binary language.
